Para agregar el comando en la consola de Mac, usar el comando 
- Si esta instalado zsh
    ~~~
     cat mkrepo >> ~/.zshrc
    ~~~
    Despues usar 
    ~~~
      nano .zshrc 
    ~~~
    Ver que modificaciones hacer respecto a los paths
- Si usas bash
  ~~~
    cat mkrepo >> ~/.bash_profile
  ~~~
  Despues usar
  ~~~
    nano .bash_profile
  ~~~
  Ver que modificaciones hacer respecto a los paths
